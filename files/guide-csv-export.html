Available Exports
🚨 Incident Export
HIBCW_active_YYYY-MM-DD.csv  /  HIBCW_archived_YYYY-MM-DD.csv
All reported incidents including suspect descriptions, vehicle details, and map coordinates. Three views: Active, Archived, and All.
Admin Panel → Incidents → ⬇ Export CSV
👥 Member Export
HIBCW-Members-YYYY-MM-DD.csv
All approved members with contact info, business details, district assignment, and days active in the program.
Admin Panel → Members → ⬇ Export CSV
📋 Business Registry Export
HIBCW_BusinessRegistry_YYYY-MM-DD.csv
Full registry of all businesses (all statuses) including approval dates, packet sent status, and Captain assignments. Respects active search filters.
Admin Panel → Registry → ⬇ Export CSV
📊 Annual Tracker Export
HIBCW_Tracker_YYYY.csv
Monthly statistical summary for the selected year — incidents, members, alerts, captain logins, and application counts.
Admin Panel → Statistics → Tracker → ⬇ Export CSV

Incident Export — Column Reference
ColumnTypeDescription
IDUUIDUnique identifier for the incident — matches the database record. Use to cross-reference with Supabase.
TypeTextIncident category: Theft/Shoplifting, Burglary, Robbery, Vandalism, Vehicle Break-In, Drug Activity, Fraud/Scam, Suspicious Activity, Other
StatusTextactive — visible on public map  ·  archived — removed from map, retained for records
Date/TimeTimestampWhen the incident occurred (not when it was submitted), formatted in local time.
AddressTextStreet address or nearest intersection as entered by the reporter.
DescriptionTextFree-text narrative from the reporter. May contain line breaks; quoted in the CSV.
ReporterTextReporter's name if provided. Many reporters submit anonymously — this field will be blank.
Police ReportTextHPD report number if the reporter filed one. Blank if no report was filed or unknown.
Suspect Height … Suspect OtherTextSeven suspect description fields. All optional — blank if not observed or not entered.
Vehicle Make … Vehicle NotesTextSix vehicle fields. All optional — blank if no vehicle was involved or details unknown.
Lat / LngDecimalMap pin coordinates. Blank if the reporter only typed an address without confirming a pin location.
Media CountIntegerNumber of photos/videos attached to the incident. Media files are stored in Supabase Storage.
SubmittedTimestampWhen the report was submitted to the system (may differ from Date/Time of the incident).

💡 Tips for Working with HIBCW CSV Exports
📅
Dates are local Hawaiʻi time. Timestamps exported from the Incidents view are formatted in your browser's local timezone. For UTC values, query Supabase directly.
🔤
Text fields are double-quoted. Any field that may contain commas (Description, Address, Notes) is wrapped in quotes with internal quotes escaped as "".
🔍
Registry export respects filters. If you have a search term or district filter active when you click Export CSV, only the visible rows will be exported — not the full dataset.
🗂
Opening in Excel? Use Data → From Text/CSV and set encoding to UTF-8. This preserves Hawaiian diacriticals (ā, ʻ, ō) in district and place names.
